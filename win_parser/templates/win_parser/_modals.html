Network Adapters

×
Guidance

This section will give you insight into:
1. The IP address of the host (most helpful when trying to make sense of the network when cross-referencing network diagrams and firewall configurations).
2. How many "Network Interfaces" this host is connected to. Usually, this aligns with the "Network Interface Controllers" (NIC). For example, this could represent the number of ethernet cables plugged into a desktop computer and the WiFi receivers/adapters used on the machine
3. See if the system is connected to a Wireless Network, indicated with a "Wireless LAN adapter" parameter with an IP address registered.
Note: The table skips and does not show any adapters with the value of "Media disconnected".
